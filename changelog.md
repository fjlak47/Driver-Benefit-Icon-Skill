# Changelog

## v5

- 基于用户调色后的参考图更新颜色与明度基线。
- 新增 `DBI-00-MAT-002_v1_high-purity-bright-clean-color-reference.png`。
- 明确当前核心风格为 `Bright Clean 3D UI Icon Style / 明亮清爽 3D UI 图标风格`。
- 进一步移除旧版 `果冻感 / 糖胶感 / 半透明软糯感` 表述。
- 新增色彩规则：高纯度红橘、纯橙、明亮黄、奶白高光、浅暖阴影。
- 新增避免项：偏灰橙、脏红、褐橙、土黄、暗金、肉粉感、浑浊渐变。
- 强化“放大精致、缩小清晰”的目标。
- 强化复杂度限制：总语义元素不超过 3 个，结构层级不超过 3 层。

## v4

- 新增 Bright Clean 3D UI Icon Style / 明亮清爽 3D 图标风格。
- 去掉原来的果冻感 / 糖胶感 / 半透明软糯感。
- 强化：明亮、清爽、solid、结构清楚、边界清晰、轻阴影、干净高光。
- 新增原则：切忌元素和分层结构过多过复杂。
- 新增目标：图标必须兼顾放大和缩小两类场景。
