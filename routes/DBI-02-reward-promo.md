# DBI-02 / Reward Promo / 奖励营销型

## Style

Inherit **Bright Clean 3D UI Icon Style / 明亮清爽 3D UI 图标风格**.

## Purpose

Used for reward, subsidy, bonus, membership benefit, promotion, and conversion-oriented scenarios.

## Visual Direction

- warmer and brighter
- stronger benefit feeling
- high-purity red-orange, pure orange, bright honey yellow, creamy white
- clean highlights and light shadows
- no jelly, no gummy, no muddy gradients
- not cluttered

## Complexity

- Main object: 65%–70%
- Supporting object: 25%–30%
- Accent: 5%
- Total semantic objects: no more than 3
- Structural layers: no more than 3
