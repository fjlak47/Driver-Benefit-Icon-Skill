# References

Put visual reference files in this folder.

Recommended files:

```text
style-reference.png
approved-icons.png
anti-patterns.png
```

## style-reference.png

放置已有司机端触点图标风格合集，用于稳定图标家族感。

## approved-icons.png

放置已经确认可用的图标结果，用于后续生成时做正向参考。

## anti-patterns.png

放置不推荐样式，例如：

- 元素太多
- 过度真实
- 过度复杂
- 不适合小尺寸
- 背景太重
- 文字或 logo 太明显
- 图标不像同一套风格
